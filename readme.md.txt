# Employee Management System

A Python-based Employee Management System using MySQL for database management.

## Features

* Add Employee
* View Employees
* Update Employee Salary
* Delete Employee
* Menu-driven interface

## Technologies Used

* Python
* MySQL
* mysql-connector-python
* Git & GitHub

## Database Schema

employees table:

* emp_id (Primary Key)
* name
* department
* salary

## How to Run

1. Create the MySQL database and employees table.
2. Install dependencies:

pip install mysql-connector-python

3. Update database credentials in the Python file.
4. Run:

python employee_management_system.py

## Project Outcome

This project demonstrates CRUD operations, database connectivity, and menu-driven application development using Python and MySQL.
